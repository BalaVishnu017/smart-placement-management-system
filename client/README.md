# Smart Placement Management System - Frontend Client

This directory contains the React.js client frontend for the Smart Campus Placement Management System (SPMS), built using Vite.

## 🛠️ Tech Stack
- **React.js** (Vite environment)
- **React Router DOM** (Single page routing)
- **React Icons** (Dashboard iconography)
- **Axios** (Centralized HTTP client configuration)
- **Context API** (Global state management for Auth, Toast notifications, etc.)

## 🏃 Quick Start

1. Install client-side dependencies:
   ```bash
   npm install
   ```
2. Start the Vite local development server:
   ```bash
   npm run dev
   ```
3. The application will be running at `http://localhost:5173`. Make sure the backend server (on port `5000`) is running concurrently so that API endpoints resolve correctly via Vite's proxy configurations.

For full project instructions, configurations, and features, please refer to the [Root README](../README.md).
